# App Design Folders

This directory provides app-scoped design specs under a unified AVRAI design system.

- `consumer_app`: Primary user discovery/social journey
- `business_app`: Business operations and growth workflows
- `admin_desktop_app`: Admin/ops monitoring and controls
- `research_portal`: Analytics and research-facing workflows
- `partner_sdk_examples`: Partner integration-facing UX surfaces

All app folders must comply with `docs/design/DESIGN_REF.md`.
All feature-level packs must map to `docs/design/MASTER_PLAN_DESIGN_LINKAGE.md`.
Cross-system/degraded/recovery UX states must map to `docs/plans/architecture/REALITY_COHERENCE_TEST_MATRIX.md` (`RCM-UX-019`, `RCM-UX-020`).
