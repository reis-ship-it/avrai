## Placement Rule

Phase `7.9` is the autonomous research/evidence loop.

- Use `7.9` for: hypothesis generation, experiment contracts, evidence scoring, and promotion governance.
- Use owning phases for implementation:
  - memory/data architecture -> Phase 1.x
  - training/runtime compute -> Phase 5.x
  - planner/inference behavior -> Phase 6.x
  - lifecycle rollout/rollback -> Phase 7.7/7.7A
  - device-tier behavior -> Phase 7.5
  - federated propagation -> Phase 8.1
  - cross-system robustness policy -> Phase 10.9.x
