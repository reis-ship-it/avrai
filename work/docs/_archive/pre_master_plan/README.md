# Pre-Master-Plan Documentation Archive

**Archived:** February 2026

## What's Here

Documentation created before the world model master plan was established (February 2026). These files are preserved for historical reference but are **no longer the source of truth** for any active development.

## Archive Policy

Files are moved here when they meet ANY of these criteria:

- **Superseded by Master Plan** -- The master plan (`docs/MASTER_PLAN.md`) now defines the execution sequence for all active development
- **Historical fix/session logs** -- One-time fix documentation, session progress reports, or compilation fix logs that served their purpose
- **Completed phase reports** -- Reports from phases that are fully complete and no longer need active reference
- **Pre-world-model documentation** -- Plans and docs written before the world model architecture was defined
