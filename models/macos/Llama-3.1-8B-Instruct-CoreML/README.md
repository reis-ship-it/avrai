# Model Card for Model ID

This model is a converted version of Meta's Llama-3.1-8B-Instruct model to CoreML format using the [llama-to-coreml](https://github.com/andmev/llama-to-coreml) project.

## Model Details

Meta's Llama-3.1-8B-Instruct model is a 8-billion parameter large language model that is based on the Llama-3.1 architecture. It is a fine-tuned version of the Llama-3.1-70B model for instruction following.
